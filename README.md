# Speech Enhancement

## Set up in you machine

- Clone the repo

```bash
mkdir ~/Dev/speech-enhancement -p
cd ~/Dev/speech-enhancement
git clone https://github.com/arvind-4/speech-enhancement.git
```

- Create a python virtual env

```bash
cd ~/Dev/speech-enhancement
uv init --python=3.7
```

- Install Dependencies:

```bash
uv pip install -r requirements.txt
```

Open the `src/notebooks` and run any notebook!
